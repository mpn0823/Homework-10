# Note Taker

Web app that lets you save notes

## Table of Contents

- [Usage](#Usage)
- [Installing](#Installing)
- [Authors](#Authors)
- [License](#License)

## Usage

- Notes are displayed in left sidebar. 
- Floppy icon saves notes. 
- Trashcan icon deletes notes. 
- Pencil icon clears text.
